┌────────────────────────────────────────────────────────────────────────────────┐
│ CUSTOMIZATION GUIDE                                                            │
└────────────────────────────────────────────────────────────────────────────────┘

TO MODIFY RANGES:
1. Open data_generator.py
2. Find the weight_ranges dictionary (line ~38)
3. Modify the tuple values: ('min', 'max')
4. Save and run

Example:
  weight_ranges = {
      'M10': (8.100, 8.400),  # Changed from 8.300
      'M15': (8.100, 8.350),  # Changed from 8.300
  }

TO CHANGE NUMBER OF ROWS:
1. When prompted, enter desired row count
2. Or modify num_rows=1000 parameter in code

TO REBUILD EXECUTABLE:
  python3 build_exe.py
